To demonstrate how we can change the class of an object with javaScript, lets look at a new example. This next example will turn an object from a box into a circle. It will do this by changing the CSS class of the object. Want to see how that works? Let's take a look!

JAVASCRIPT - CircleClass.js

function cir(x) {
	
	if (x.className == "BoxA"){
		x.className = "BoxB";

	}else if(x.className == "BoxB"){
		x.className = "BoxA";
		
	}
}
HTML

<!DOCTYPE HTML>
<html lang="en-us">
<head>
<style> 
.BoxA {
  background-color: green;
  height: 100px;
  width: 100px;
  left:0px; top:0px;
  border-radius: 2px;
}
.BoxB {
  background-color: blue;
  height: 100px;
  width: 100px;
  left:0px; top:0px;
  border-radius: 55px;
}
</style>
<meta charset="utf-8">
<title>BoxToCircle</title>
<script src="circleClass.js"></script>
</head>

<body>
	<div class="BoxA" id="ShapeChange" onClick="cir(this)"></div>
</body>
</html>

Lesson 8

Creating CSS3 Transitions and Animations in CSS and triggering them with JavaScript

Let us first look into Triggers and Transitions. The code you are about to look at is for a simple button that changes color each time it is pressed and when the mouse is hovered/not hovered over it. The button triggers a change in the CSS class of the object back and forth and could just as easily be tied into an animation if we feel like it. To keep it simple, we will post the animation after this. You will notice the javaScript is similar to the example in the previous lesson. Turns out it's a good simple way to flip flop styles.

JavaScript - text.js

function change(x) {
	
	if (x.className == "text1"){
		x.className = "text2";
	}else if(x.className == "text2"){
		x.className = "text1";
	}
}
HTML

<!DOCTYPE HTML>
<html lang="en-us">
<head>
<style> 
.text1 {
    color: black;
}
.text1:hover {
	color: #66ffff;
	background: #0066cc;
}
.text2 {
    color: red;
}
.text2:hover {
	color: #ffff66;
	background: #cc6600;
}
</style>
<meta charset="utf-8">
<title>Mondays</title>
<script src="text.js"></script>
</head>

<body>
	<div align="center"><h2 id="text" class="text1" onclick="change(this)">Random innocuous statement about how mondays are the worst.</h2></div>
</body>
</html>

Let's see that in action then shall we? Click on it and watch the magic happen. It has a hover effect and a click change, which is worth mentioning.

That was fun. Now let's move on to Animations, which are just as fun! First, we need to cover a basic concept, that of KeyFrames. KeyFrames are basically motion points for an object, which make the animation process a lot easier because we don't need to design every single frame of the object. We specify a start point, some optional middle points, and an end point and CSS does the rest. Pretty nifty really. This next one was a lot of fun, it is literally a sphere that runs away from your mouse! Cool right? Let's look at an example!

JavaScript - sphere.js

function move(x) {
	
	if (x.className == "circle transform"){
		x.className = "circle transform-active";
	}else if(x.className == "circle transform-active"){
		x.className = "circle transform";
	}
}
HTML and CSS Style

<!DOCTYPE HTML>
<html lang="en-us">
<head>
<style> 
.circle {
  background-color: green;
  height: 100px;
  width: 100px;
  left:0px; top:0px;
  border-radius: 55px;
}

.transform {
	background-color:green;
  -webkit-transition: 3s;
  -moz-transition: 3s;
  -ms-transition: 3s;
  -o-transition: 3s;
  transition: 3s;
  margin-left: 0% !important;
}

.transform-active {
	background-color:blue;
  -webkit-transition: 3s;
  -moz-transition: 3s;
  -ms-transition: 3s;
  -o-transition: 3s;
  transition: 3s;
  margin-left: 93% !important;
} 
</style>
<meta charset="utf-8">
<title>Chase the Sphere!</title>
<script src="color.js"></script>
</head>

<body>
	<div class="circle transform" id="sphere" onmouseover="move(this)"></div>
</body>
</html>

Alright, catch that sphere! It will run away from you so you gotta be fast!.

to start with, let's have some fun with rotation! That's something we have yet to cover and so it will be a fun little avenue of exploration. the program below is a circle that flips over when you hover your mouse on it. Neat huh? Next, we're going to look at a Animations! We did a really awesome one last lesson, but I think we need to add a new one for this one so let's have fun with other stuff!. For the sake of simplicity, I went with an animation that just goes and goes. I did this because when I was first learning about animations I didn't realize that you could have them move indefinitely. I've gone with a nice pleasing box for our purposes here. Since no JavaScript is used in this, I'll show the CSS and then give an example of the animation itself.

CSS

div {
    width: 100px;
    height: 100px;
    background-color: black;
    position: relative;
	animation-iteration-count: infinite;
    animation-name: animated;
    animation-duration: 4s;
}
@keyframes animated {
    0%   {background-color:blue; left:0px; top:0px;}
    50%  {background-color:black; left:1230px; top:0px;}
    100% {background-color:blue; left:0px; top:0px;}
}

Kinda soothing, I might just fall asleep on my keyboard! Next, we're going to look at a Transition! For this we will replicate a hover effect.

JavaScript - color.js

function alternate(x) {
	
	if (x.className == "shape1"){
		x.className = "shape2";
	}else if(x.className == "shape2"){
		x.className = "shape1";
	}
}
HTML and CSS Style

<!DOCTYPE HTML>
<html lang="en-us">
<head>
<style> 
.shape1 {
    width: 300px;
    height: 100px;
	border-radius: 10px;
    background: #1111FF;
	transition: background 2s;
}

.shape1:hover {
	background: #FF1111;
}
.shape2 {
    width: 300px;
    height: 100px;
	border-radius: 10px;
    background: #112255;
	transition: background 2s;
}

.shape2:hover {
	background: #11FF11;
}
</style>
<meta charset="utf-8">
<title>Shapes N Stuff</title>
<script src="color.js"></script>
</head>

<body>
	<div class="shape1" onclick="alternate(this)"></div>
</body>
</html>
